RS Seating Chart Planner -
Fixed Seating Visualization for Odoo

Visual fixed-seating planner for Odoo with drag & drop on SVG floorplans.

RS Seating Chart Planner is a visual seating module for Odoo that lets you define buildings, floors, and rooms, upload SVG plans, and assign users to seats using a drag & drop interface.
Using a clean interface and Odoo’s native views, this module helps HR, facility teams, and receptionists quickly locate and display where employees are seated.


Key Features:

SVG floorplans per location
Drag & drop seating assignment
Avatar pop-ups with user info
Hierarchical locations (building → floor → room)
Quick search for users
Admin-only placement control
Open Source (AGPL-3.0)

Use Cases:
  • Onboarding: help new hires find team members
  • Facility planning: assign assets by location
  • Security & safety: know who is seated where
  • Visitor reception: quick visual lookup
