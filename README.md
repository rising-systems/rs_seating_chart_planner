# Rising Systems Seating Chart Planner

**Rising Systems Seating Chart Planner** is an Odoo module designed to visually manage seating and people placement using SVG-based plans. It enables intuitive planning and assignment of people to spaces using an interactive map-based interface.

![App Icon](rs_seating_chart_planner/static/description/icon.png)

## 📦 Module Structure

This repository includes the following Odoo modules:

- **`rs_seating_chart_planner`** – Main module for visual seating and assignment management.
- **`rs_locations_manager`** – Supporting module for managing physical locations and their hierarchies.

## 🔗 Dependencies

- The **Seating Chart Planner** depends on the **Locations Manager** module, which is included in this repository.
- Both modules are compatible with Odoo Community and Enterprise editions.
- No external Python libraries are required.

## 🧪 Testing

Run Odoo tests as usual:

  ```bash
  odoo-bin -c odoo.conf --test-enable --stop-after-init
  ```
